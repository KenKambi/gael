## 📦 **Installation**  
1. **Clone the repo:**  
   ```bash  
   git clone https://github.com/KenKambi/gael.git  
   ```  
2. **Install dependencies:**  
   ```bash  
   npm install  
   ```  
3. **Launch dev server:**  
   ```bash  
   npm start  
   ```
