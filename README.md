## Getting Started

### Prerequisites

- Node.js and npm installed

### Installation

```bash
git clone https://github.com/KenKambi/gael.git
cd gael
npm install
```

### Running the Project

```bash
npm run dev
```

Open your browser and navigate to the local server URL (typically `http://localhost:5173/`).
